# 学生打卡签到系统sign_in.py
 
基于 Python 控制台开发的轻量化签到管理工具，纯原生代码、无需第三方库，实现学生签到登记、记录查询、人数统计、数据持久化保存，适合Python新手练手。
 
## 项目介绍
 
本项目是简易班级签到管理小程序，利用 字典键唯一特性 防止重复签到，所有数据自动保存到本地  sign.txt  文件，重启程序数据不丢失。操作简单、逻辑清晰，适合巩固文件读写、字典操作、异常捕获等基础语法。
 
## 核心功能
 
- ✅ 学生签到：录入姓名+签到时间，自动拦截重复签到
- ✅ 全员名单查看：格式化展示所有签到人员与对应时间
- ✅ 单人签到查询：快速检索指定学生是否签到
- ✅ 删除签到记录：支持删除指定人员签到数据，实时同步文件
- ✅ 签到人数统计：一键统计当前总签到人数
 
## 文件结构

## 运行方式
 
1. 下载项目所有文件至同一文件夹
2. 直接运行  sign_in.py 
3. 根据菜单序号选择对应功能即可操作
 
## 数据存储格式
 
采用  #  作为分隔符存储数据，格式规范清晰：
姓名#签到时间
示例：张三#08:30 
 
## 学习核心知识点
 
1. 字典特性：利用 Key 唯一性，天然防止重复签到
2. 字典遍历解包： for name, t in data.items()  批量读写键值对
3. 字典长度统计： len()  快速实现人数统计
4. 本地文件持久化：读写 TXT 文件，数据永久保存
5. 自定义分隔符：使用  #  分割结构化文本数据
6. 异常捕获：拦截非法输入，避免程序闪退
